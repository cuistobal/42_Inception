Projet d'Administration Système réalisé par chrleroy
Infrastructure Docker multi-services avec Docker Compose

📋 Services Déployés

NGINX

Serveur web avec TLSv1.2/1.3, point d'entrée unique sur le port 443.

WordPress

CMS avec PHP-FPM, configuration automatisée et cache Redis intégré.

MariaDB

Base de données pour WordPress avec utilisateurs configurés.

Redis

Cache pour améliorer les performances de WordPress.

FTP Server

Serveur vsftpd pointant vers les fichiers WordPress.

Adminer

Interface de gestion de base de données légère.

Portainer

Interface de gestion Docker pour monitorer les containers.

Static Website

Site statique Python - vous y êtes ! 🎉

Tic Tac Toe (Minimax)

Jeu de morpion avec une IA utilisant l'algorithme Minimax — l’IA explore les coups possibles et choisit l’option optimale. Cliquez ici pour jouer et afficher le code JavaScript.

Docker Docker Compose NGINX WordPress PHP-FPM MariaDB Redis vsftpd Adminer Portainer Python SSL/TLS Debian
